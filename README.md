# Customer-segmentation-dashboard

## 📌 Project Overview

This project demonstrates how to build an interactive **Customer Segmentation Dashboard** in **Google Sheets** using retail sales transaction data. The dashboard combines **Pivot Tables**, **RFM (Recency, Frequency, Monetary) Analysis**, charts, and slicers to analyze customer behavior, identify valuable customer segments, and provide actionable business insights.


## 🎯 Objectives

- Clean and organize sales transaction data.
- Perform customer analysis using Pivot Tables.
- Segment customers using the RFM model.
- Build an interactive dashboard with charts and slicers.
- Analyze sales trends, customer behavior, and marketing performance.
- Support data-driven business decisions.

---

## 📂 Dataset Information

The dataset contains retail sales transaction records with the following fields:

| Column | Description |
|--------|-------------|
| Transaction ID | Unique transaction identifier |
| Customer ID | Unique customer identifier |
| Purchase Date | Date of purchase |
| Product Category | Product category purchased |
| Items Quantity | Number of items purchased |
| Unit Price | Price per item |
| Total Amount | Total purchase value |
| Payment Method | Payment method used |
| Shipping Country | Customer's shipping country |
| Marketing Channel | Customer acquisition source |
| Customer Type | New or Returning customer |
